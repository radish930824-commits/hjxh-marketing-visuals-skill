# 营销视觉制作流程

## 1. 收集需求

确认渠道、尺寸、目标受众、营销目标、产品、文案、必须使用的素材和允许的图片处理。缺少尺寸时读取 `channel-sizes.md`，推荐 1-3 个选项并确认。

保留用户原文。文案过长、层级不清或不适合版面时，提供精简版本并等待确认；不得擅自修改价格、优惠、产品声明或商业事实。

## 2. 判断任务等级

- 日常且信息完整：直接制作一个推荐方向。
- 重要活动、方向不明确或用户要求探索：先提供 2-3 个差异化方向。
- 涉及高风险修改：必须先确认。

高风险包括 Logo、品牌色、字体、IP 身份或新动作、产品事实、促销价格和条款。

## 3. 选择素材

从 `assets/manifest.json` 自动选择匹配的 Logo、IP 动作和产品图。素材含义不明确、需要新 IP 动作或可能影响产品真实性时，先询问用户。

Logo 缺失时停止。产品或标准 IP 素材不足时，报告限制；经确认后使用占位方案或制作明确标记的“待审核草案”。

## 4. 选择工具

- Canva：真实素材组合、准确文字排版、可编辑交付。
- ImageGen：创意背景、新场景或环境延展。
- 组合：先用 ImageGen 生成不含 Logo 的背景或场景，再在 Canva 中加入真实素材、Logo 和文字。

## 5. 制作和交付

保持一个主要视觉焦点，产品优先于 IP，促销红色仅小面积使用。输出前执行 `review-workflow.md`；任一 MUST 违规必须修正或报告“不通过”。

交付内容按任务类型提供：可编辑 Canva 设计与导出图片，或 ImageGen 图片、制作说明与审核结果；方案任务提供构图、素材、文案层级、尺寸、工具建议和提示词。
