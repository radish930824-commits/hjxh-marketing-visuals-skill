# 营销图片渠道尺寸

用户指定尺寸时严格执行。未指定时，根据用途推荐 1-3 个选项并等待确认。

| 渠道或用途 | 推荐尺寸 |
|---|---|
| 微信朋友圈竖图 | `1080 x 1440 px` |
| 微信朋友圈方图 | `1080 x 1080 px` |
| 微信公众号头图 | `900 x 383 px` |
| 小红书竖图 | `1242 x 1660 px` |
| 电商商品主图 | `800 x 800 px` |
| 电商活动横幅 | `1920 x 600 px` |
| 通用社交方图 | `1080 x 1080 px` |
| 通用竖版海报 | `1080 x 1440 px` |
| 通用横版宣传图 | `1920 x 1080 px` |

推荐时说明选择理由，例如信息量、阅读场景、平台裁切风险和是否需要多尺寸适配。
