# 皇家小虎品牌规范

## 品牌核心

- 品牌名称：皇家小虎
- 品牌口号：全球美食私厨
- 视觉印象：鲜明、亲和、活力、美食感、圆润
- 品牌主体：头戴皇冠的黄色小虎

## 标准颜色

| Token | HEX | 用途 |
|---|---|---|
| `brand.yellow` | `#FFCC00` | 主背景与品牌识别 |
| `brand.brown` | `#632C16` | 标题、正文与深色元素 |
| `brand.red` | `#D93924` | 小面积促销与关键强调 |
| `neutral.white` | `#FFFFFF` | 留白与浅色背景 |

MUST 使用准确色值。小虎红不得替代小虎黄成为大面积主背景，除非用户明确批准。

## IP

- 结构参考：`ip-specification`
- 欢迎、庆祝：`ip-action-clap`
- 推荐、认可、品质背书：`ip-action-thumbs-up`
- 自信站立、鼓励、品牌宣言：`ip-action-confident`
- 问候、服务引导：`ip-action-wave`
- 活力、行动号召：`ip-action-run`

IP 必须保持 Q 版、潮玩感、3D 公仔风格和大头小身体比例。头部约占整体高度 `60%-65%`。不得改变脸部、皇冠、虎纹、尾巴、身体比例、标准色或角色气质。新动作或新视角必须标记为“待审核草案”或“非标准动作”。

## 产品真实性

| 产品 | 资产别名 |
|---|---|
| 披萨 | `product-pizza` |
| 烤肠 | `product-sausage` |
| 蛋挞 | `product-egg-tart` |
| 手抓饼 | `product-scallion-pancake` |

MUST 优先使用真实产品图。允许保持真实性的裁切、缩放、抠图和背景延展。MUST NOT 虚构配料、夸大份量、改变结构、成熟度、颜色或质感。

## 视觉语言

- 使用圆润、清晰、亲和的视觉形态与充足留白。
- 优先使用小虎黄、白色与小虎咖。
- 食品图片保持明亮、清晰、有食欲且自然。
- 禁止阴暗冷酷、金属感、赛博朋克、尖锐攻击性或过度复杂的主视觉。

## 待确认默认规则

Logo 净空与最小尺寸、颜色比例、字号与行高、场景构图属于保守默认值。正式对外输出时如有争议，应由品牌负责人确认。
